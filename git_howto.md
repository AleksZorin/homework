# Подсказка по GIT

## Создание локального репозитория (инициализация)
~~~sh
git init
~~~

## Добавление версионности (внесенных изменений) файла в репозитории
~~~sh
git add
~~~

## Фиксация версии и уведомление о внесенных изменениях
~~~sh
git commit -m "Messege"
~~~
